01 Â· Principles

Five rules of decision.

These are not aesthetic preferences. They are filters: if a component breaks one of these, it doesn’t ship. Inherited from V11 hiring research R5 (voice), R7 (craft), R8 (anti-patterns).

01

Taste-first, not role-first.

The landing opens with a point of view on agentic design, not a rÃ©sumÃ© summary. A hiring manager at Anthropic/Linear/Cursor reads the first sentence as a thesis â€” and either leans in or closes the tab.

02

Profile + agents = combination.

Never “Danilo is a solo genius leveraging AI”. Always “I keep the craft, the agents give me leverage.” This rule filters headlines, case copy, and testimonials.

03

Show the artifact running.

If we claim agentic, a real agentic artifact is embedded above the fold. PDF renders in-page, the generator streams output, a skill chip expands into its definition. Claim + evidence, not claim alone.

04

Restrained accent, not decorative.

Orange #FF8964 is punctuation, not atmosphere. No warm-radial pill glow (Huly anti-pattern). Used on one primary CTA, one accent tick per stat, one focus ring, one chip border.

05

Self-hosted fonts, WCAG AA, reduced-motion, skip link, hreflang, LCP < 1.5s. The landing IS the work sample â€” if a tier-1 reviewer spots a craft leak in 20 seconds, nothing else matters.

09 Â· Accessibility

Floor: WCAG AA. Ceiling: keyboard-only review clean.

The landing IS the work sample. A tier-1 reviewer will tab through it in 30 seconds and close it if a focus ring disappears. Five non-negotiables from V11 R7:

Floor 01

Self-hosted fonts, preloaded

Inter (variable, Latin-1), JetBrains Mono, Source Serif 4 served from assets/fonts/. No third-party CDN. font-display: swap. LCP font preload link in <head>.

Floor 02

Skip link + landmarks

First focusable element is <a class="v11-skip">. <header> / <nav> / <main id="main"> / <footer> all labeled. No invalid role="marquee".

Floor 03

Focus ring always visible

:focus-visible shows 2px accent outline with 3px offset on every interactive element. Never removed, never “cleaner” without.

Floor 04

Reduced-motion honored

All animations collapse to 0ms under prefers-reduced-motion: reduce. Live-dot and caret freeze in idle state, never hidden.

Floor 05

Hreflang + lang attributes

Root lang matches rendered language. rel="alternate" hreflang="en|es|x-default" links in both directions. Language switch preserves scroll + section.

Floor 06

Keyboard nav tests

Tab traverses: skip â†’ brand â†’ tabs â†’ lang toggle â†’ cta â†’ essay TOC â†’ cases. Enter activates, Escape closes modals. No focus trap.
