cross-cutting insights for dr-cv

What AI-landings are ALL doing right now

Patterns that appear across 5+ of the 9 sites. Treat as the current "ante" — doing less than this will read as pre-2024.

  1. The word "agent(s)" has moved into the H1. Linear, LangChain, Inkeep ("teammates"), Cursor (subhead) all use it. dr-cv take: "Agentic Designer" in the H1 — but pair it with an outcome verb, not a noun claim.
  2. Single restrained accent color, zero gradient soup. Each site has one dominant hue (cyan, indigo, blue, terracotta) and treats it like a signature. Orange (#FF8964) for dr-cv reads fresh because nobody in the agentic space owns orange — Anthropic's terracotta is the closest neighbor.
  3. Product screenshot / IDE mock is the hero's protagonist. Cursor, Linear, v0, Replicate all put a real interface in the hero. Editorial-only heroes (Anthropic, LangChain) are the exception, not the rule. dr-cv take: put a real skills-sheet PDF or a real CV composer output in the hero frame.
  4. Mono typeface earns its keep. Where mono appears (Cursor IDE demos, Replicate code snippets, Raycast ⌘-keys, Vercel CLI) it signals "real engineering happens here." JetBrains Mono for dr-cv section labels + code-fragment testimonial metadata is on-trend.
  5. Long-scroll + sticky header is universal. 9/9 sites. No tab-SPAs, no horizontal-scroll gimmicks. Pricing is link-only (only Replicate has an inline table).
  6. Motion is scarce and surgical. Only Raycast has a true WebGL/Three.js hero (the cube). Everyone else uses Framer-Motion-style layout transitions, pre-rendered screen recordings, or static SVG. dr-cv take: one animated motif (dotted grid or type-in terminal) is enough — don't over-index on shader theater.
  7. "Generated with / powered by / teammates" framing. Inkeep ("AI Teammates"), Linear ("teams and agents"), v0 ("Prompt. Build. Publish."). The narrative is leverage and collaboration, not "AI replaces you." dr-cv take: "profile + agentic AI = combination" wording Danilo already uses is aligned with the category.
  8. Primary CTA is short + imperative. Download · Get started · Start deploying · Start building · Try Claude. Two or three words, action verb first. Never "Learn more as primary."
  9. Dark-theme is not automatic for AI. 6 of 9 lead with light. Dark is Linear and Vercel — both have established brand equity to ground it. For a new personal brand, light + one accent is the safer signal of seriousness.
  10. Editorial H1 vs utility H1 is a real fork. Anthropic ("safety at the frontier") and Replicate ("Run AI with an API.") are the poles. Danilo's positioning sits closer to Anthropic's editorial register — but the page needs Replicate-style proof (real artifacts, real numbers) below the fold.
