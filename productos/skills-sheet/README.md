# productos/skills-sheet

Skills sheet de 2 páginas A4.

## Variantes

- `dist/skills-sheet-en.{html,pdf}`
- `dist/skills-sheet-es.{html,pdf}`

## Cómo generar

```
npm run build:skills-sheet
```

## Fuente de datos

- `data/skills.yaml` (taxonomía dual: por capa de producto + por outcome de cliente)
- `data/identity.yaml`, `data/experience.yaml`
- Plantillas en `generators/templates/skills-sheet*.ts`
