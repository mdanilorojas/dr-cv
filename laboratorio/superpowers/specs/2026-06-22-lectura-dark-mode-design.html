Arquitectura de Archivos

  • Parser: generadores/lib/parse-lectura.ts parsea el archivo MD de lectura a JSON estructurado.
  • Template: generadores/templates/lectura.ts define la estructura HTML + CSS del blueprint y el script cliente.
  • Generator: generadores/v11-landing.ts importa el parser y la template para emitir el HTML final.
