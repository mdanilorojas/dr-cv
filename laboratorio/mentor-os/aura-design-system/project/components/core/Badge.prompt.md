Badge is a compact status or metadata pill; `mono` switches to the uppercase JetBrains Mono treatment for codes and versions.

```jsx
<Badge variant="success" dot>live</Badge>
<Badge mono>v11.r10</Badge>
```

Variants: `neutral` · `primary` · `accent` · `success` · `warning` · `danger` · `info`. For skill mastery use the dedicated `Mastery` component instead — those colors are sacred.
