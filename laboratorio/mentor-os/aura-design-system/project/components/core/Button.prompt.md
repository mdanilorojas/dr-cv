Button is the primary action control for the Aura app — flat accent fill, 36px default, no scale-on-press and no glow.

```jsx
<Button variant="primary">Earn $250,000</Button>
<Button variant="accent">Run diagnostic</Button>
<Button variant="ghost" size="sm">Cancel</Button>
```

Variants: `primary` (flat accent), `accent` (hairline ring), `secondary`, `ghost`, `danger`. Sizes: `sm` · `base` · `lg`. Use `iconOnly` for square icon buttons, `block` for full-width. The accent is punctuation — reserve `primary` for the one true action per view.
