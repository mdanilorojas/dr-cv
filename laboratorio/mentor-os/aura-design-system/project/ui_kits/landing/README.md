# danilorojas.design — Landing UI Kit

A high-fidelity recreation of the Aura **landing / notebook surface** — a designer's essay, not a marketing page.

## Sections
- **Nav** — monogram, name, anchor links, primary CTA.
- **Hero** — notebook thesis headline (JetBrains Mono) + serif lead, beside a **live artifact** (the CV being tailored: embed well → stream → plan → skill chips).
- **Proof strip** — four mono metrics in a hairline grid.
- **Thesis** — the cream "paper" chapter: Source Serif 4 at a 42rem measure with an accent drop-cap.
- **The artifact** — the prose-grader surface embedded inline (stream + plan + diff + mastery).
- **CTA** — portrait, outcome-first headline, dual actions.

## Rhythm
Dark → (cream paper) → dark. The cream chapter is the only light surface; it signals "reading," everything else is the product. The accent stays punctuation throughout — no hero glow, no gradients.

Everything composes the compiled bundle (`window.AuraDesignSystem_58f91b`); the kit adds only layout and copy.
