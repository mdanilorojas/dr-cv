Take the diagnostic · prose

Answer 96 questions in your own words

Each question maps to one skill. You type the answer the way you'd say it in an interview — no multiple choice. Claude scores it on four dimensions and returns a number, a one-line critique, and a diction note on your English.

3
Scores set levels

Skills fill in poco a poco

One or more questions determine each skill's level — gap → novice → developing → proficient → mastered. The chips fill as you pass. Every level banks a fraction of that skill's dollar value into your live market number.

4
Generate the study program

Gaps become an ordered roadmap

Failed and untested questions are priced as "locked value." The program orders modules by the money each unlocks, prerequisites first — so you always study the thing that moves the number most.

Learn → re-answer → level up

Close the loop

After studying, re-answer the question. A higher score levels the skill up, banks more value, and the dashboard's market number climbs toward $250k. Repeat until the gap is zero.

C

The prose grader

In canvas it runs on the preview helper. In production it moves behind a server route — the key never touches the browser.

Canvas (now)

// screen-assessment.jsx · ExamRunner.score()
const raw = await window.claude
  .complete(prompt);
// → {substance, structure,
//    english, concision,
//    overall, feedback, diction}

Production (Vercel)

const r = await fetch('/api/score', {
  method: 'POST',
  body: JSON.stringify({ answer, q })
}).then(r => r.json());
// api/score.ts → Anthropic SDK
// ANTHROPIC_API_KEY server-only

Four dimensions — the reason it's prose, not multiple choice

  • Substance (.45) — is the answer correct, complete, senior-level?
  • Structure (.20) — claim → why → trade-off, well reasoned?
  • English (.20) — fluency, grammar, natural phrasing for a US/EU room.
  • Concision (.15) — says it tightly, no padding.
